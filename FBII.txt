	USO DEL SISTEMA FBII, OMNI 400

-> PARA ACTIVAR EL SISTEMA ES NECESARIO QUE TODAS LAS AREAS PROTEGIDAS 
   ESTEN CERRADAS Y QUE EL INDICADOR DE LISTO ESTE ENCENDIDO, PEDIRLE QUE 
   PRECIONE SU CLAVE Y EL IDICADOR DE ARMADO SE ENCENDERA Y CONTARA DE 30 
   SEGUNDOS PARA SALIR DE SU INMUEBLE.

-> PARA ACTVAR Y QUEDARSE EN SU INMUEBLE DEJE PRECIONADA LA TECLA STAY 5 SEGUNDOS Y EL 
   SISTEMA SE ACTIVARA DE IMEDIATO.

-> PARA ACTIVAR EN MODO INSTANTANEO ELIMINANDO EL TIEMPO DE RETARDO EN LA PUERTA 
   DE ENTRADA PRECIONE LA TECLA DE INSTANT O INSTANEO EN ESTE MODO LOS DETECTORES
   DE MOVIMIENTO QUEDAN FUNCIONANDO.

-> PARA ACTIVAR EN MODO INSTANTANEO Y QUEDARSE DENTRO DE SU INMUEBLE PRECIONE LA 
   TECLA DE INSTANT Y STAY Y EL SISTEMA QUEDA TRABAJANDO PERO ELIMINA LOS DETCTORES 
   DE MOVIMIENTO.

-> PARA DESACTIVAR SU SISTEMA SOLO PRECIONE SU CLAVE DE 4 DIGITOS Y EL SISTEMA QUEDARA 
   DESACTIVADO.

-> PARA BLOQUEAR UNA ZONA PRECIONE LA TECLA BYPASS Y LA ZONA QUE DESE DESACTIBAR EJEMPLO:
   SI LA ZONA QUE QUIERE DESACTIVAR ES LA 2. PRESIONE LA TECLA BYPASS 2 Y EL INDICADOR DE 
   LA ZONA 2 EMPESARA A FLASHEAR ESTO INDICA QUE LA ZONA QUEDO ANULADA.

-> PARA DAR DE ALTA CLAVE PRESIONE LA TECLA DE CODE Y SU CODIGO MAESTRO Y 
   NUMERO DE USUARIO QUE DESEE  DAR DE ALTA EJEMPLO: SI QUIERE DAR DE ALTA EL USUARIO 4.

              CODE - CLAVE – 04 - NUEVA CLAVE - 3.

-> TECLAS DE EMERGENCIA DEL SISTEMA FBII Y OMNI SON LAS MISMAS QUE A CONTINUACION LES DAMOS.

	- PANICO: 
		PRESIONE LAS TECLAS DE ( * ) Y ( # ) AL MISMO TIEMPO. 
	- MEDICA: 
		PRESIONE LAS TECLAS  DE ( 3 ) Y ( 1 ) AL MISMO TIEMPO.
	- FUEGO: 
		PRESIONE LAS TECLAS DE ( 7 ) Y ( 9 ) AL MISMO TIEMPO. 

	
	FALLAS DE SISTEMA FBII, OMNI 400

SI EL SISTEMA NO TIENE FALLA EL FOCO AMARILLO CA/BA DEBE ESTAR EN AMARILLO CONSTANTE 
ESTO INDICA QUE LA ALARMA ESTA CORRECTAMENTE Y NO TIENE FALLA.

-> SI EL FOQUITO AMARILLO ESTA APAGADO ES QUE TIENE FALLAS EN LA CORRIENTE AC
   POSIBLE QUE NO TENGA CORRIENTE LA ALARMA, O EL TRANSFORMADOR ESTE DAÑADO,  
   TENGA SUELTO EL TRANSFORMADOR.          

-> SI EL INDICADOR AMARILLO ESTA PARPADEANDO TIENE PROBLEMAS DE BATERIA DEL SISTEMA 
   ESTA FALLANDO POR DIFERENTES MOTIVOS TIENE FALLAS CON LA CORRIENTE, O LA BATERIA 
   ESTA FALLANDO.

-> SI LOS FOQUITOS DE LA PARTE DE ABAJO DEL TECLADO ESTAN PARPADEANDO ES QUE TIENE 
   UNA MEMORIA DE ALARMA Y NO LE PERMITE ACTIVAR FAVOR DE PRESIONAR LA TECLA DE STAY, 
   Y EL SISTEMA SE RESTABLECERA.

-> SI EL FOQUITO DE TX ESTA PARPADEANDO TIENE PROBLEMAS CON LA COMUNICACIÓN DE LA ALARMA 
   POR DIFERENTES MOTIVOS.
