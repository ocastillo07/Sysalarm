	MANEJO DE ADEMCO, FIRST ALERT, HOONYWELL

EL USO DE ESTOS SISTEMAS ES UN POCO DIFERENTE DE LOS DSC O TIPO POWER 
TANTO PARA ARMAR, DESARMAR, ANULAR ZONAS Y ENVIAR EMERGENCIAS.

-> ACTIVAR EN MODO AUSENTE O TOTAL, VERIFICAR QUE EL SISTEMA ESTE 
   ASEGURADO TOTALMENTE ESPERANDO QUE EL INDICADOR DE LISTO O READY 
   ESTE EN VERDE, SI ESTA EN ESE STATUS, PRECIONAR SU CLAVE Y EL NUMERO 2 
   O LA TECLA DE * - 2, Y EL INDICADOR VERDE SE APAGARA Y EL DE ARMED O 
   ARMADO SE ENCENDERA EN ROJO, TENDRA CIERTO TIEMPO PARA SALIR DEL LUGAR. 

-> PARA ACTIVAR EN STEY O PARCIAL VERIFICAR QUE EL SISTEMA ESTE ASEGURADO 
   TOTALMENTE ESPERANDO QUE EL INDICADOR DE LISTO O READY ESTE EN VERDE, 
   SI ESTA EN ESE STATUS, PRECIONAR SU CLAVE Y EL NUMERO 3 O LA TECLA DE * - 3, 
   Y EL INDICADOR VERDE SE APAGARA Y EL DE ARMED O ARMADO SE ENCENDERA EN ROJO, 
   Y PODRA ANDAR LIBREMENTE POR SU INMUEBLE.

-> PARA PODER BLOQUEAR UNA ZONA ES NECESARIO QUE PRECIONE SU CODIGO EL 6 Y EL 
   NUMERO DE ZONA QUE DESE BLOQUEAR ANTEPONIENDO EL 0 ANTES QUE EL NUMERO.    

       EJEMPLO: SI QUIERO ANULAR LA ZONA 4 PONGA SU CODIGO-6-04 
                Y EL TECLADO LE DARA CONFIRMARA CON UN BEEP Y LE DESPRENDERA 
                UNA LEYENDA QUE DICE BYPASS Y ESTARA LISTO PARA ARMAR.                          

-> PARA ACTIVAR EN MODO INSTANTANEO SIN RETARDOS Y QEDARSE DENTRO DE SU INMUEBLE 
   STAY PRECIONE * - 4 Y LE PONDRA EL TECLADO LA LEYENDA DE INSTANT Y STAY QUEDANDO 
   ELIMINADO EL DETECTOR DE MOVIMIENTO Y EL RETARDO DE LA PUERTA DE ENTRADA.

-> PARA ACTIVAR EL DOOR CHIME, ES NECESARIO QUE EL USUARIO PRECIONE SU CODIGO 9 Y 
   LAS ZONAS QUE ESTAN PROGRAMADAS INSTANTANEAS O PERIMETRALES Y PUERTAS, AL ABRIRSE 
   EMITIRA UN BEEP EL TECLADO DE SU ALARMA, SI DESEA DESABILITAR ESTA FUNCION PRECIONE 
   NUEVAMENTE SU CLAVE Y EL NUMERO 9.

-> PARA DESACTIVAR SU SISTEMA DE ALARAM ES NECESARIO QUE PRECIONE SU CODIGO Y EL NUMERO 1 
   Y EL SISTEMA QUEDA DESACTIVADO ESTE PASO LO PUEDE HACER EN CUALQUIER DE LAS FUNCIONE AWEY, 
   STAY, INSTANT, MAXIMO.

-> PARA DAR CLAVES EN EL SISTEMA ES NECESARIO QUE PRECIONE SU CLAVE 8 EL NUMERO DE USUARIO 
   Y CLAVE NUEVA EL TECLADO HARA 2 BEEP. 

       EJEMPLO: SI DA DE ALTA EL USUARIO 3 CLAVE – 8 - 3 - NUEVO CODIGO
                PARA  PROBAR SOLO PRESIONE SU CLAVE Y UNO Y EL TECLADO 
                PITARA UNA VEZ INDICANDO QUE SU CLAVE QUEDO DADA DE ALTA.

-> TECLAS DE EMERGENCIA DEL SISTEMA ADEMCO, FIRST ALERT, HOONYWELL, SON LAS MISMAS EN LOS 
   TRES EQUIPOS.

	- PANICO: 
	  PRESIONE LAS TECLAS DE ( # ) Y ( * ) AL MISMO TIEMPO Y EL SISTEMA MANDARA 
          LA SEÑAL DE PANICO A CENTRAL PARA RESTABLECER EL SISTEMA PRECIONE SU CLAVE Y 1.

	- MEDICA: 
          PRESIONE LAS TECLAS DE ( # ) Y ( 3 ) AL MISMO TIEMPO Y EL SISTEMA MANDARA 
          LA SEÑAL DE AYUDA MEDICA PARA RESTABLECER EL SISTEMA PRECIONE SU CLAVE Y 1.

	- FUEGO: 
          PRESIONE LAS TECLAS DE ( 1 ) Y ( * ) AL MISMO TIEMPO EL SISTEMA ENVIARA UNA 
          SEÑAL DE MERGENCIA PARA INCENDIO,  PARA RESTABLECER EL SISTEMA PRECIONE SU 
	  CLAVE Y 1.


	FALLAS DEL SISTEMA ADEMCO, FIRST ALERT, HOONYWELL

-> SI NO PUEDE ARMAR EL SISTEMA PREGUNTAR SI TIENE EN EL TECLADO UN NUMERO DE ZONA Y 
   ABAJO LA LEYENDA DE ALARM, SI ES ASI QUE PRECIONE SU CODIGO Y 1, EN DADO CASO DE 
   QUE NO PIDALE QUE PRECIONE SU CODIGO Y 1, ESTO PARA BORRAR LAS MEMORIAS DE ALARMA 
   QUE PUEDA TENER SU EQUIPO.

-> SI LE MARCA EN EL TECLADO (FC) ES QUE TIENE EN SUS SISTEMA FALLA DE COMUNICACIÓN, 
   Y ES POR FALLAS DIVERSAS: TELEFONO SUSPENDIDO, CUENTAN CON INTERNET INFINITUM, O 
   PERDIDA DE LINEA.

-> EL SISTEMA TIENE FALLA DE CORRIENTE MARCARA NO AC,  NO CA, NO RED, Y ESTO ES POR 
   CORTE DE CORRIENTE, FALLA DE TRANSFORMADOR.

-> SI LE MARCA (OC) EL SISTEMA TIENE PROBLEMAS CON EL CABLEADO DEL TECLADO AL CONTROL 
   NECESITA SERVICIO.

-> CUANDO TIENE FALLA DE BATERIA EL TECLADO EMITIRA UN BEEP Y LE APARECERA UNA LEYENDA DE BAT, 
   POSIBLE QUE AYA TENIDO FALLA CORRIENTE O LA BATERIA ESTE FALLANDO.

-> SI EL TECLADO ESTA PITANDO Y LE APARECE CHEK ZONE, UNO DE LOS TRASMISORES TIENE BAJA LA 
   BATERIA ES NECESARIO EL CAMBIO DE LAS MISMAS, O UNOS DE TRANSMISORES ESTAN FALLANDO.
